# 世界观校准回路

> 需要时加载：`read skills/knowledge-base-habits/reference/worldview-calibration.md`

## 校准触发条件

- 连续 2 周的日记中，方向相关行动占比 < 30%
- 用户主动表达「我最近感觉偏离了」「不确定方向对不对」
- 遇到一个事件，让之前的方向锚定显得不够用了

## 校准三步

```
Step 1: 诊断
  → 对比 MEMORY.md 的方向锚定 vs 近期日记中的行动记录
  → 指出偏离点（不评判，只描述事实）

Step 2: 追问
  → 「这个偏离是因为精力不够，还是因为方向本身需要调整？」
  → 用户回答 → 记录到所以为人/层

Step 3: 更新
  → 如果方向变了 → 更新 MEMORY.md + 所以为人/层笔记
  → 如果方向没变 → 在日记中记录恢复计划 + Tasks
```

🔴 CHECKPOINT：校准后的决定必须用户亲口确认，晴川不能替用户改方向。
