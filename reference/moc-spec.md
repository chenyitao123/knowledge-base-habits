# MOC 规范参考

> 需要时加载：`read skills/knowledge-base-habits/reference/moc-spec.md`

## MOC 层级

```
README
  → 层 MOC（工具/🧭 · 人/🧭 · 品牌/🧭）
      → 专项 MOC（如 XX研究_MOC）
      → 子目录索引（如 领域A/🧭）
```

| 层级 | 是什么 | 内容 |
|------|--------|------|
| README | 总入口 | 世界观 + 层 MOC 跳转 |
| 层 MOC | 阅读地图 | 2-4 条阅读路径（「读完你会」）、跨层连接、专项 + 索引列表 |
| 专项 MOC | 单命题阅读地图 | 聚焦一个命题的多条阅读路径 |
| 子目录索引 | 内容索引 | 按类别列出目录有什么，不写阅读路径 |

> 专项 MOC 和子目录索引同级，都在层 MOC 之下。

## 层 MOC 模板

```markdown
---
area: [工具/人/品牌]
status: evergreen
created: YYYY-MM-DD
tags: [MOC, ...]
---

# [层名] · 认知地图

> **这条路径帮你回答**：[一句话]

## 📖 阅读路径

### 路径一：[主题]
> 适合：[场景]

- [[笔记A]] — 干什么用
- [[笔记B]] — 干什么用
→ 读完后你会：[收获]

## 🔗 跨层连接
- 向上/向下/向内/向外 → [[其他层MOC]]

## 🧭 连接
- 家 → [[README]]

## 📊 子目录索引
- [[子目录A/🧭]] · [[子目录B/🧭]]
```

## 子目录索引模板

```markdown
---
status: evergreen
created: YYYY-MM-DD
---

# [目录名] · 索引

> 这个目录有什么

### [类别一]
- [[笔记1]] — 一句话说明
- [[笔记2]] — 一句话说明

### [类别二]
- [[笔记3]] — 一句话说明

## 🧭 连接
- 家 → [[层MOC]]
```

## 创建规则

| 触发 | 建什么 |
|------|--------|
| 层目录存在（工具/人/品牌） | 必须建层 MOC（永远只有三个） |
| 目录下笔记 ≥ 3 篇 | 建子目录索引（仅限层下一级子目录，更深层由父索引覆盖） |
| 单命题有 ≥ 3 篇跨目录关联笔记 | 建专项 MOC（与子目录索引同级）。阅读路径只连本域笔记，跨领域走 🧭 连接区指针 |

## 反例

- ❌ 子目录索引写成 MOC（加了阅读路径）
- ❌ 专项 MOC 的阅读路径里列举其他领域的单篇笔记——跨领域是指针，不是导游
- ❌ MOC 做成字母排序的文件清单
- ❌ 目录 1-2 篇就建索引
- ❌ 层 MOC 没有跨层连接
- ❌ 笔记变了 MOC 不更新
- ❌ 层 MOC 和子目录索引之间没有双向链接
